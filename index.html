Automating Recruitment with Multi-Agent AI

Revolutionizing Candidate Screening & Shortlisting

Problem Overview

HR teams spend excessive time manually reviewing job descriptions and CVs.

This results in delays, inconsistencies, and missed talent opportunities.

Need a system that can automate: JD reading, CV matching, shortlisting, and interview scheduling.

Our Multi-Agent Solution

JD Reader Agent

Intelligently analyzes job descriptions to extract key requirements and qualifications.

CV Parser Agent

Efficiently processes and extracts relevant information from candidate resumes.

Matching Agent

Uses advanced AI to match candidate profiles with job requirements.

Shortlisting Agent

Ranks and shortlists the best candidates based on multiple criteria.

Communication Agent

Handles automated communication with candidates and interview scheduling.

Technologies Used

  • NLP: spaCy, BERT (HuggingFace Transformers)
  • Matching: Sentence-BERT + cosine similarity
  • Database: MongoDB
  • Backend: Python + FastAPI
  • Agent Communication: LangChain
  • Email API: SendGrid

Data Flow
